## Running from source

Dota 2ls requires [Node.js](https://nodejs.org/) and [Electron.js](https://electronjs.org/)

Install the dependencies:

```sh
cd dota2ls
npm i
npm test (or electron .)
```

###### Or try to compile with electron packager
